本项目用于模拟和分析新新魔塔的战斗伤害的数学期望，并可用蒙特卡洛模拟验证结果

## 主要功能
- 提供蒙特卡洛模拟与理论公式解对比
- 自动化批量测试与统计输出
- 支持普通怪物，特殊属性：坚固、仿攻、魔攻、连击
- 支持勇士和怪物的暴击、闪避、攻击/防御临界值
- 支持徽章（如霸者之证3级）等多种效果


## 文件说明
- `damage.py`：主程序，包含模拟器类，测试函数
- `test.ipynb`：包含了测试的样例，测试方法参考文件中的写法即可
